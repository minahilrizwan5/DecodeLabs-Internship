# 🤖 Python Rule-Based Chatbot

A simple chatbot built using Python that responds to user inputs using predefined rules. This project demonstrates basic programming logic, condition handling, and simple conversational flow.

---

## 🚀 Features
- Beginner-friendly chatbot
- Rule-based response system
- Text-based interaction (CLI)
- Easy to understand and run
